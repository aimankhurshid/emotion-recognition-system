‚ùå Limitation of Base Paper

Static Spatial Analysis Only

The base DCD-DAN architecture treats facial expressions as static snapshots, analyzing spatial features (eyes, mouth, eyebrows) at a single time point. This misses the temporal dynamics of how real expressions unfold over time.

Example: In "Surprise", eyes widen first (t=1), then mouth opens (t=2). Static methods can't capture this critical sequential pattern.

‚úÖ Our Bi-LSTM Enhancement

Temporal Sequential Modeling

We add Bidirectional LSTM layers after dual-attention to model how facial features activate sequentially. Bi-LSTM learns both forward (eyes‚Üímouth) and backward (mouth‚Üíeyes) temporal dependencies.

Technical Win: LSTM memory cells maintain context across feature sequences, detecting subtle temporal patterns that distinguish similar emotions (Fear vs. Surprise: +3.2% improvement).
